# Attribution and provenance

## SBLGNT

The Greek text used in the upstream artifacts is ultimately the SBL Greek New Testament. The Faithlife/SBLGNT repository identifies the SBLGNT as a critically edited Greek New Testament and lists license information as Creative Commons Attribution 4.0 International.

## MorphGNT

This patch includes selected MorphGNT source snapshots from `morphgnt/sblgnt`. MorphGNT documents the token columns as book/chapter/verse, part of speech, parsing code, text, word, normalized word, and lemma. It states that the SBLGNT text itself is subject to the SBLGNT EULA and the morphological parsing and lemmatization is available under CC-BY-SA.

## Prior generated artifacts

This package also copies prior generated artifacts from this conversation into `inputs/previous_artifacts/` and records SHA-256 hashes in `MANIFEST.yaml`.
