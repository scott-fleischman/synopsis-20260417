# Attribution and provenance

This package includes generated analysis code and derived data created in this conversation.

Upstream source bases used:

- Faithlife / SBLGNT repository for Greek text files and apparatus XML.
- MorphGNT SBLGNT edition for token-level morphology and lemmas.
- Prior thread artifact registries for the Thomas/apocrypha-curated layers reused in the JTEA patch.

Important licensing note:

- SBLGNT is **not public domain**. It is identified in the repository README as **CC BY 4.0**.
- MorphGNT states that the SBLGNT text is subject to the SBLGNT EULA and that the morphological
  parsing / lemmatization is under **CC-BY-SA**.

This patch does not attempt to supersede upstream licenses; it records the local provenance and
attribution needed to interpret the rerun package.
