# Attribution and provenance

This supplement is derived from source snapshots and prior artifacts generated in the same project thread.

## Ultimate textual sources

- SBL Greek New Testament (SBLGNT), Faithlife/SBLGNT. The repository identifies the SBLGNT as CC BY 4.0 and lists version history through v1.2.
- MorphGNT SBLGNT. The MorphGNT README states that the SBLGNT text is subject to the SBLGNT EULA and that the morphological parsing and lemmatization are available under CC-BY-SA.
- Coptic Scriptorium corpora for Coptic Thomas data as represented in the prior JTEA package.
- M. R. James, *The Apocryphal New Testament*, used only through the prior apocrypha inventory layer; this supplement does not newly ingest apocryphal text beyond Thomas.

## Redistribution note

The package includes derived/source-snapshot CSVs from prior generated artifacts for auditability. Any public redistribution should preserve attribution and verify that the downstream data files satisfy source-specific license requirements.
