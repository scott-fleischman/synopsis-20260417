# Updated conclusions after high-priority supplement

## 1. Markan-source model is better supported than before

The missing Mark–Luke comparison was the largest gap. The new Mark–Luke package shows that Luke, like Matthew, has a substantial Mark-like narrative alignment. This strengthens the broader claim that Mark or a Mark-like written narrative source belongs near the center of the Synoptic model.

Caution: Luke has extensive non-Markan material and a common-tradition model remains live in some local blocks.

## 2. John should not be visually subordinated

John now has pairwise John↔Mark, John↔Matthew, and John↔Luke datasets. The result is not a continuous chain but a set of anchor episodes and local lexical cases. The best global model remains shared tradition/anchor dependence rather than simple copying of one Synoptic Gospel.

## 3. Thomas and epistles need case-first presentation

The Thomas matrix confirms that logion-level analysis is the right unit. Epistle cases should be read as dossiers, not as a heatmap first. 1Tim 5:18 / Luke 10:7 remains strongest; 1Cor/Luke Eucharist remains high formula-risk; James remains likely Jesus-tradition evidence without demonstrated direct Matthew/Luke copying.

## 4. Visualization priorities should change

The site should now lead with conclusion/evidence navigation and the Gospel square, then Mark–Luke and John pairwise. The older Synoptic-only top navigation is no longer adequate.
