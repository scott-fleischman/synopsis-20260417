# Executive summary

This supplement adds the missing high-priority data layers requested after the GitHub repository review.

## Mark ↔ Luke

The direct Mark–Luke package now exists. It uses the same general Synoptic alignment logic as Mark–Matthew but treats Mark 16:9–20 text-critically: the long ending is excluded from the default primary chain and retained as a sensitivity condition.

Default Mark–Luke results:

- 183 primary-chain pairs.
- 35 loose blocks and 76 tight blocks.
- 140 secondary echoes at score ≥ 0.22.
- Burden totals:
  - Mark prior / Luke uses Mark-like source: 254.6466
  - Luke prior / Mark uses Luke-like source: 303.1966
  - Shared narrative/oral tradition without direct use: 164.1468

Interpretively, this supplies the missing control for the Markan-source claim: Luke does show substantial Mark-like local alignment, though with extensive Lukan non-Markan material.

## John pairwise

John is now exposed as first-class pairwise data:

- John↔Mark: 33 candidate pairs score ≥ 0.25; 10 anchors.
- John↔Matthew: 33 candidate pairs score ≥ 0.25; 10 anchors.
- John↔Luke: 34 candidate pairs score ≥ 0.25; 9 anchors.

The pairwise result supports shared tradition/anchor analysis rather than a continuous copying chain.

## Thomas and epistles

Thomas now has a logion × Gospel matrix. Epistle–Gospel evidence now has case dossiers, with formula risk separated from retrieval strength. The strongest high-value epistle saying case remains 1Tim 5:18 / Luke 10:7.
