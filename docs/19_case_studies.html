Case studies

Ten high-value cases with full Greek, token alignment, and per-hypothesis burden

Each case pairs a source (usually an epistle or John) with a target (usually a Synoptic Gospel). The full Greek texts are shown with source and target colored; the token alignment below renders the same texts as an edit script (equal, insert, delete, replace-a / replace-b) so you can see exactly how the two differ. Each hypothesis card carries the analyst's narrative estimate of its burden. The provisional best explanation is the model the case most strongly supports — not a final verdict.
