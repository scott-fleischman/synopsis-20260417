per-unit, per-direction, per-model obligation ledger (1,945 rows from atlas)

The atlas enumerates 102 pericope-level units (92 computational Synoptic blocks and 10 John anchor episodes) and attaches a per-model, per-direction obligation to each: a required explanation, the contrary evidence against that direction in this unit, and the best response model available. This browser lets you pick a unit and see exactly what each system model must explain there.

How to read: each unit card first lists its ranges across the four gospels and its methodological_status (these are computational alignments, not traditional pericope titles). Below the ranges, obligations are grouped by direction, then by model. Direction IDs link to the per-direction dossier (page 37); model IDs link to the system-model scorecard (page 39). A unit with no John range is a Synoptic-only alignment; John-anchor units show the anchor name in the status line.
