The Gospel of Thomas survives in Coptic (NHC II,2) with three Greek fragments (P.Oxy. 1, 654, 655). Because this package parses the Coptic, direct Greek-to-Greek dependence is not automatically scored. Instead the registry is a hand-curated mapping: a Thomas logion → canonical motif → direct-dependence burden → best-fit transmission model. The honest default for most logia is shared sayings tradition with independent crystallization; Thomas frequently sharpens toward revelatory/esoteric framing while Matthew/Luke frame the same saying for broader paraenesis.
