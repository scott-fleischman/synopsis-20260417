Low-verbatim package

John · Thomas · NT epistles · apocrypha — a network-model sister package to the Synoptic analyses

This package steps outside the Synoptic backbone into material where verbal agreement is thinner, sayings migrate independently, and "parallels" are often thematic rather than verbatim. Unlike mm / ml / mld, a single monotonic order-chain is the wrong tool here. Instead the package uses a multi-layer network model: exact Greek retrieval, windowed sayings matching, negative-space flagging for scripture/liturgy false positives, curated semantic registries for John, Thomas, James, and the Pastorals, and a hypothesis-burden ledger that tracks which tradition-transmission model best fits each case.

The six data layers

Stacked from raw Greek surface forms to curated semantic judgment. Later layers get progressively more human-reviewed; earlier layers are automatic and noisy but exhaustive.

The five hypothesis-burden classes

For each potential literary dependency, one of these five transmission scenarios is usually the best fit. The burden ledger quantifies how large the lift is to defend each scenario for a given case.
