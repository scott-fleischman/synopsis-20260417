Gospel Square

the 4-gospel relationship matrix · Mark · Matt · Luke · John

A single matrix view of every pairwise Gospel relationship the workbench covers — which pairs have dedicated analysis, which have retrieval-only coverage, and which are unexplored. Each cell is clickable for drill-down to the specific evidence stream that grounds it.

The diagonal (self-cells) is not a relationship — it carries self-stats (verse counts, chapters). Off-diagonal cells carry pairwise evidence. Because our analyses are directional in some packages and symmetric in others, each off-diagonal cell describes what the data shows for that ordered pair; read the detail panel for what direction the underlying package actually supports.
Click a cell to see the evidence stream that grounds that pair.

John × Synoptic anchor episodes

Shared narrative episodes where John and one or more Synoptic Gospels converge. Each entry records the shared features + how each direction of dependence would have to transform the material if it were prior.
