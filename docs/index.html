How to read

Colors

  • In Mark↔Matt: Mark is brick-red, Matthew slate-blue.
  • In Matt↔Luke: Matthew takes the "source" red, Luke the "target" blue.
  • Score heat: cream (0.1) → umber (1.0).
  • Secondary echoes: teal.

Units

  • Primary chain — one monotonic 1-to-1 alignment.
  • Secondary echo — a non-primary high-similarity pair.
  • Micro / tight block — adjacent high-confidence pairs.
  • Macro / loose block — a larger coherent aligned span.
  • Nonlocal echo — a strong match not positioned on the chain.
