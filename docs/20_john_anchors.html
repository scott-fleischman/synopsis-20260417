John / Synoptic anchor episodes

Ten shared narrative scenes, reviewed as anchors rather than verse-aligned pairs

John shares a core set of narrative moments with the Synoptics — the Baptist witness, a temple action, a feeding, a walk on water, an anointing, an entry, an arrest, a denial, the crucifixion and garments — but each is transformed so thoroughly that verse-level alignment usually fails. The honest analytic unit is the anchor: a shared episode whose structural features can be compared even when the lexical texture diverges. For each anchor below, the registry records what the episode shares, how John would transform it if the Synoptics were prior, how Synoptics would transform it if John were prior, a shared-tradition assessment, and a provisional best explanation. These are structured reasoning prompts, not final verdicts.
