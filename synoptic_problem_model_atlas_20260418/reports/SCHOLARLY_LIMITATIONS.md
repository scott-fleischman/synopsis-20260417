# Scholarly Limitations

1. The pericope ontology is computational and anchor-based; it is not a replacement for a critical synopsis.
2. The minor-agreement catalog is a screening layer. It should be hand-reviewed before being used in formal argument.
3. The double-tradition order catalog depends on the Markan mask. Multiple mask regimes should remain visible.
4. John is deliberately treated differently from the Synoptics. John/Synoptic evidence is anchor-based and often low-verbatim.
5. Burden scores are audit scores, not probability estimates.
6. Some components inherit prior generated artifacts. This package bundles those inputs and declares them, but it does not turn every upstream generated artifact into a raw-source-only reconstruction.
7. Text-critical sensitivity is flagged, not fully adjudicated.
8. The model scorecard is meant to clarify tradeoffs, not to mechanically select a winning model.
