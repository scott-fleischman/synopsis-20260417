# Remaining Work for an Even Stronger Release

The atlas is comprehensive as a model-comparison dataset, but an even stronger scholarly release would add:

- a hand-curated traditional pericope synopsis crosswalk;
- a hand-reviewed canonical minor-agreements catalog with classification by type;
- full variant-sensitive reruns for high-value exact-match loci;
- multiple Markan-mask regimes incorporated directly into the model scorecard;
- full phrase-level Greek side-by-side rendering for every pericope dossier;
- external scholarly review of response-model prose;
- DOI archival release with versioned source commitments.
