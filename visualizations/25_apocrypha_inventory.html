Apocrypha inventory

M. R. James corpus — inventory layer, not full primary-language comparison
Inventory only · not analysis No retrieval has been run across these works. This page is a scope registry: M. R. James\u2019 catalogue, grouped by corpus, with an expected_signal note per work describing what intertext would plausibly surface if primary-language alignment were done. Nothing on this page is an analytic finding about any apocryphal work. For the one exception — Thomas — see 21 Thomas parallels and 32 Thomas × canon.

This is the scope layer only, not a lexical analysis. The analyses elsewhere in this package work on Greek (SBLGNT) and Coptic (Coptic Scriptorium Thomas), which are the texts we have in forms amenable to automatic retrieval. The broader apocryphal corpus — infancy gospels, apocryphal acts, passion and resurrection accounts, Gnostic tractates — is enumerated here as a registry of what would require primary-language alignment work in a follow-up pass. Each entry carries an analysis_layer flag marking how deep the current pipeline reaches, and an expected_signal note describing what intertext would plausibly surface with proper alignment.

Future-analysis pointers. To carry the methodology from the canonical pipeline into the apocryphal corpus, three layers are needed: (1) a primary-language text base for each work (Greek fragments for Protoevangelium, Acts of Paul and Thecla, etc.; Coptic NHC tractates; Latin Acts of Pilate); (2) a stable reference/segmentation scheme, since apocryphal works lack the verse numbering that makes Gospel/epistle retrieval windowable; (3) alignment of sayings to the Thomas/canonical registry so that verbal or thematic overlap can be scored against a shared substrate. The present package stops at (0) — inventory — because going further without proper primary-language preparation would produce noise, not evidence.
