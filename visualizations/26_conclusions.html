Three-way confidence

A single confidence grade smuggles three very different questions into one column. This table splits them out — a case can be lexically strong but directionally uncertain (e.g. 1 Cor 11 / Luke 22 has the longest exact run in the package but Paul pre-dates Luke, so direction runs the opposite way of any naïve read).

Retrieval — does the automatic pipeline (score, exact-run, n-gram) see a signal at all?
Interpretive — once a signal is seen, how confident is the reading we built on top of it?
Directional — can we tell which way any dependence runs (A→B, B→A, shared source)?
