Mask-regime sensitivity

How the Q-core size shifts under strict / medium / broad masking (reproducibility-patch rerun).

The reproducibility patch runs three masking regimes against the same similarity matrix: strict (threshold 0.40), medium (0.355, the project default), and broad (0.325). A tighter threshold keeps more verses in play, so the residual Q-core is larger; a looser threshold masks more aggressively, so the residual Q-core is smaller. The question this page answers is: how fragile is the Q-core count against mask-threshold choice?
